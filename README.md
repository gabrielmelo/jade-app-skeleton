# JADE APP SKELETON

Jade app Skeleton is a basic front-end structure to create HTML template with Jade.

```
    |       |-- views
    |           |-- 404.jade
    |           |-- about.jade
    |           |-- home.jade
    |           |-- single-aside.jade
    |           |-- single.jade

```
